# 📚 Research Paper Search & Summarization Agent

An AI-powered application that enables users to search, retrieve, and summarize research papers using **Retrieval-Augmented Generation (RAG)** and **Large Language Models (LLMs)**. The project leverages modern AI frameworks to provide accurate, context-aware summaries and improve the research workflow.

---

## 🚀 Features

* 🔍 Intelligent research paper retrieval
* 📄 AI-generated summaries of research papers
* 🧠 Retrieval-Augmented Generation (RAG)
* 📚 Vector search using FAISS
* 🤖 Powered by Groq (Llama 3.3)
* 🔗 LangChain-based workflow
* ⚡ Fast and efficient semantic search

---

## 🛠️ Tech Stack

* **Programming Language:** Python
* **Framework:** LangChain
* **LLM:** Groq (Llama 3.3)
* **Vector Database:** FAISS
* **Embeddings:** Hugging Face Embeddings
* **Notebook:** Jupyter Notebook

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/your-username/research-paper-search-summarization-agent.git
```

### 2. Navigate to the project directory

```bash
cd research-paper-search-summarization-agent
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

---

## ▶️ Usage

1. Open the Jupyter Notebook.
2. Configure your **Groq API Key**.
3. Load the research papers.
4. Generate embeddings using Hugging Face.
5. Create the FAISS vector index.
6. Ask questions related to the uploaded research papers.
7. Receive AI-generated, context-aware summaries.
